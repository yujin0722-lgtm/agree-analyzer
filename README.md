# AGREE II 評価集計ダッシュボード

AGREE IIフォーム作成ツールで出力した「ダッシュボード用CSV」を読み込み、領域別スコア、評価者間不一致、使用推奨判定、コメントを表示する静的HTMLダッシュボードです。

## 使い方

1. `index.html` をブラウザで開きます。
2. AGREE IIフォーム作成ツールで作成した `dashboard_agree2_*.csv` を読み込ませます。
3. 画面上で結果を確認します。
4. 必要に応じて「結果をPDFで保存」から印刷・PDF保存します。

## GitHub Pagesで公開する場合

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内の `index.html` をリポジトリ直下にアップロードします。
3. GitHubの Settings → Pages から公開設定を行います。
4. 公開URLにアクセスし、手元のCSVを読み込ませて使用します。

## 注意

- Excelファイル（`.xlsx`）は直接読み込めません。必ずCSVファイルを使用してください。
- CSVの解析はブラウザ内で行われます。
- GitHub Pages上には回答CSVを置かない運用を推奨します。評価者氏名や自由記載コメントが含まれる可能性があるためです。
- このHTMLはTailwind CSS、Chart.js、Lucide、Google FontsをCDNから読み込みます。

## 推奨ワークフロー

AGREE IIフォーム作成ツール → Googleフォーム → 回答スプレッドシート → ダッシュボード用CSV → このダッシュボード
